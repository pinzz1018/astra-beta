Gemini API Key

Masukkan API key Gemini kamu (format AQ.Ab... atau AIza...). Key disimpan hanya di perangkat ini, tidak dikirim ke server manapun selain langsung ke Google.

Groq API Key

Opsional: masukkan API key Groq untuk memakai model Llama 3.1 8B Instant (lebih cepat/murah). Key disimpan hanya di perangkat ini.

OpenRouter API Key

Opsional: masukkan API key OpenRouter untuk memakai model Astra 2.0 (Qwen3 Coder) dan Astra 2.5 (NVIDIA Nemotron 3 Super) — keduanya gratis. Key disimpan hanya di perangkat ini.

remove.bg API Key

Opsional: masukkan API key remove.bg untuk memakai fitur ganti/hapus background foto (gratis 50 kali/bulan). Key disimpan hanya di perangkat ini.

Belum punya key? Buat gratis di aistudio.google.com/app/apikey

Groq key gratis di console.groq.com/keys

OpenRouter key gratis di openrouter.ai/keys

remove.bg key gratis (50x/bulan) di remove.bg/dashboard#api-key
